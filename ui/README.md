# ui

All in-game UI scenes and their scripts. Does not include main menu (that lives in core/).

- `hud/` — heads-up display during gameplay (health, resources, cooldowns)
- `pause_menu/` — pause overlay with resume/settings/quit buttons
- `loading_screen/` — loading bar/spinner between levels
- `tooltips/` — hover tooltips for items, enemies, players (world-space labels)
- `inventory/` — inventory / equipment screen, item drag-drop logic
- `menus/` — settings panel and other overlay menus
